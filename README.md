### 🚀 Features at a Glance

*   **⚡ Any Input:** Paste text, upload PDFs/DOCXs, snap a photo (OCR), or paste a YouTube link.
*   **🧠 Deep AI Analysis:** Extracts complex words instantly, providing definitions, synonyms, and context.
*   **🌍 51-Language Translation:** Translates words, highlights cultural nuances, and detects "Mirror Words" (cognates).
*   **🎮 3 Smart Quiz Modes:** Standard, Reverse, and an AI-generated "Fill-in-the-blank" test that changes every time.
*   **💡 Memory Hooks & Stories:** AI generates funny mnemonics and weaves your vocabulary into a short story to help them stick.
*   **🗣️ Pronunciation Coach:** Speak into your mic and get graded on a 0-100% native match.
*   **💬 Socratic Tutor:** Slide-out AI chat to ask questions or request analogies about your passage.
*   **🤯 "Explain Like I'm 5" & Opposite Day:** Rewrite dense text simply, or rewrite it entirely in antonyms.
*   **📚 Cloud Library:** Save your sessions to Firebase and pick up where you left off.

### ⚙️ Quick Start

**1. Clone & Install**
```bash
git clone https://github.com/your-username/decipher.git
cd decipher
npm install
```

**2. Setup Keys**
*   Create a `.env` file with `GEMINI_API_KEY=your_key`
*   Add your Firebase `serviceAccountKey.json` and `firebase-config.js`

**3. Run**
```bash
npm start
# Opens at http://localhost:3000
```
